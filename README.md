=========
Proof of Concept Bidirectional E2E Everest Implementation

Environment
-----------
e2everest should run properly on Windows, OSX, and various distributions of Linux

Dependencies
------------
* Oracle Java 6, 7, or 8
* Maven 3.0.x

Installation
------------
`mvn package -DskipTests=true`
